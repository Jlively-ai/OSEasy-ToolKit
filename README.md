# OsEasy-ToolKit - Python Edition

> Python 重写版 - 更强大、更易维护的噢易学生端解锁工具

## 📋 版本对比

| 项目 | Batch 原版 | Python 重写版 |
|------|-----------|--------------|
| 代码行数 | ~768 行 | ~1400 行 |
| 可读性 | 较低（Batch 语法限制） | 较高（面向对象，模块化） |
| 扩展性 | 差 | 好（模块化设计） |
| 跨平台 | Windows 仅 | Windows（核心功能依赖 Win32） |
| 依赖 | 无 | Python 3.8+（零外部依赖） |
| **用户界面** | **控制台菜单（ASCII）** | **GUI 图形界面 (Tkinter) + 控制台双模式** |

## 🚀 快速开始

```bash
# 方式1：启动 GUI 图形界面（默认，推荐）
python main.py

# 方式2：强制启动控制台版本
python main.py --cli
```

### 系统要求

- **操作系统**: Windows（需要管理员权限）
- **Python**: 3.8 或更高版本
- **依赖**: **零外部依赖**（仅使用标准库 Tkinter）

> 💡 由于您的学校机房网络不佳，本工具采用 **纯原生 Tkinter** 构建 GUI，无需下载任何第三方库！

## 🎯 功能列表

### 🔓 解锁功能
- [x] **一键解锁所有限制** — 同时解锁网络、控屏、键鼠、USB
- [x] **单独解锁网络限制** — 停止 MMPC/OeNetlimit 服务
- [x] **单独解锁控屏限制** — 终止 ScreenRender 进程 + 备份文件
- [x] **单独解锁键盘鼠标锁** — 删除锁驱动 + 终止相关进程
- [x] **单独解锁USB限制** — 删除 easyusbflt 服务

### ⚙️ 管理功能
- [x] **服务管理** — 停止/启动/重启服务，批量停止所有
- [x] **进程管理** — 终止单个进程或全部进程，监控模式
- [x] **挂起/恢复 Student.exe** — 暂停或恢复运行
- [x] **持续监控模式** — 循环终止关键进程（防复活）

### 📊 状态监控
- [x] **设备信息** — 计算机名、操作系统、CPU、IP 等
- [x] **学生端状态** — 关键进程运行状态指示灯
- [x] **解锁状态** — 已解锁/未解锁/部分解锁 智能判断
- [x] **详细执行日志** — 命令、输出、结果实时显示

## 🔧 GUI 特性

- ✅ **纯 Tkinter 构建** — 零外部依赖，无需联网下载
- ✅ **5 个选项卡布局** — 状态、解锁、服务管理、进程管理、关于
- ✅ **多线程操作** — 点击按钮不卡界面
- ✅ **详细日志输出** — 命令、输出、结果分色显示（深色终端风格）
- ✅ **服务/进程状态实时显示** — 运行状态带颜色指示
- ✅ **设备信息自动获取** — 计算机名、系统、CPU、IP 等
- ✅ **解锁状态智能判断** — 根据关键进程自动判断已解锁/未解锁/部分解锁
- ✅ **后台线程防卡死** — 所有系统操作在后台执行
- ✅ **GitHub 可点击链接** — 关于页面直接跳转

## 📝 日志输出详情

所有系统操作都会在底部日志区域详细显示：

| 日志类型 | 颜色 | 示例 |
|---------|------|------|
| `[执行]` 操作描述 | 🔵 蓝色 | `[执行] 停止服务 MMPC` |
| `[命令]` 实际命令 | 🟣 紫色 | `[命令] sc stop MMPC` |
| `[输出]` 命令输出 | ⚪ 灰色 | `[输出] [SC] ControlService 失败 1062` |
| `[错误]` 错误输出 | 🔴 红色 | `[错误] 拒绝访问` |
| `[返回]` 返回代码 | ⚪ 白色 | `[返回] 代码 0` |
| `【操作】` 模块标题 | 🔵 蓝色 | `【一键解锁】开始执行全部解锁操作...` |
| `✓ 成功` | 🟢 绿色 | `服务 MMPC 停止成功` |
| `⚠ 警告` | 🟡 黄色 | `进程未运行，跳过` |
